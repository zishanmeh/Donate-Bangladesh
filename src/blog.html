Important FAQ

What is the Document Object Model (DOM)?

Document object model represents the structure of a web page as a tree of objects, allowing developers to interact with and manipulate the content, structure, and style of a webpage using programming languages like JavaScript.

How do you select an element from the DOM?

There are bunch of different ways to select an element with from DOM with js. Most commonly using selector is getElementById(). Although querySelectorAll() is also highly using by programmers. Other two are getElementByTagName() and getElementByClassName() are also use preety much.

What is event delegation in the context of the DOM, and why is it useful?

One of the beautiful things in Javascript is event delegation. If you need to select multiple item in a section you don't need to select every item individually. Just select parent element which will listen for event triggered by it's children.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, I can use methods like getAttribute(), setAttribute() to set, or remove attributes. Attributes like id, src, href can be accessed or modified directly as properties. For example, `element.src = 'new-image.jpg'` changes an image's source directly.
